# EVE Online Fleet Analysis Tool

A Next.js web application for parsing and analyzing EVE Online combat logs to provide fleet-level insights into PvP engagements.

## Features

- Parse EVE Online combat logs
- Fleet session management
- Damage analysis (dealt/taken)
- Kill tracking
- Capacitor pressure monitoring
- Participant tracking

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/` - Next.js app router pages
- `src/components/` - React components
- `src/types/` - TypeScript type definitions
- `AGENTS.md` - Agent instructions for EVE weapon systems analysis

## Learn More

This project uses [Next.js](https://nextjs.org) for the framework and follows EVE Online log parsing standards.
